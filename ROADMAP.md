# Roadmap

## Phase 1 — Foundation

- [ ] Project scaffolding and CI
- [ ] Process model types (`ProcessInfo`, `ProcessState`)
- [ ] `/proc` enumeration (list all processes)
- [ ] Basic TUI skeleton

## Phase 2 — Core features

- [ ] System metrics (CPU, memory)
- [ ] Process snapshot and diffing
- [ ] Real-time process list with sorting and filtering
- [ ] Process detail view

## Phase 3 — Interaction

- [ ] Send signals to processes (SIGTERM, SIGKILL, SIGSTOP, SIGCONT)
- [ ] Process tree view (parent-child relationships)
- [ ] Search and filter by name, PID, user, state

## Phase 4 — Advanced

- [ ] Historical data and sparklines
- [ ] Resource usage graphs over time
- [ ] Configuration file support
- [ ] Plugin system for custom metrics

## Phase 5 — Polish

- [ ] Man page and help system
- [ ] Packaging (AUR, deb, rpm, Flatpak)
- [ ] Accessibility (screen reader support, high contrast themes)
